# TangoDisplay User Guide

TangoDisplay is a native macOS menu-bar app for tango DJs. It connects to Music.app, automatically detects cortinas, and shows a clean fullscreen display on an external monitor so dancers know what genre and who is playing.

## Quick start

1. Install the app (see [Getting Started](Getting-Started))
2. Open Music.app and play a named playlist
3. Click the display icon in the menu bar › **Show Settings Window**
4. Go to **Display** and pick your external monitor
5. Click **Move Presentation Window** › **Toggle Fullscreen**

The dancer display is now live.
